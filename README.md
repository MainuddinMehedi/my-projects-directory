# What is this project?

This is Mainuddin's Project Directory. A web app that shows all the project ever built by Mainuddin Mazumder Mehedi. This web app is built for a reference purpose, just to explore the projects when I want to and when i need to find a particular project to find out how i implemented something.

# How it's organized?
First page shows the latest projects. 

All project show all the projects. 

The projects are connected through tags, technologies. Someone can click a technology and find see the proejcts done using that technology. Same for tags. 

There is a timeline page which will show the timeline of the proejcts in a year basis.
